## Summary

`diff` must be password-gated and provide text/binary diff behavior with tool integration and JSON summary.

## Acceptance Criteria

- `diff` requires password.
- Text-like files produce text diffs and binary files produce binary summaries.
- `--tool` plus standard env integrations are supported.
- `--json` summary mode is supported.

## Verification

- Validate behavior across text and binary files and with external diff tooling.

Derived from the GitSecretVault source requirements for diff behavior.
