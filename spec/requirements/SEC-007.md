## Summary

Symlink storage/restore is disabled by default.

## Acceptance Criteria

- Default lock/unlock workflows do not store or restore symlinks.

## Verification

- Validate symlink entries are blocked unless explicit allow mode is used.

## Notes

Maps directly to SEC-FS-2.
