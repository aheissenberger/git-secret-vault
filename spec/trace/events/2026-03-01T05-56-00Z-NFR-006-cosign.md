## Summary

Added cosign keyless signing to the release workflow for NFR-006.

## Changes

- `.github/workflows/release.yml`: added `sign-artifacts` job that runs after `build-local-artifacts` and `build-global-artifacts`. Uses `sigstore/cosign-installer@v3` and signs all `git-secret-vault` binaries with `cosign sign-blob --yes --bundle`. Signatures uploaded as `artifacts-signatures`. The `host` job now depends on `sign-artifacts`.
- `docs/reproducible-build.md`: appended "Artifact verification with cosign" section with `cosign verify-blob` instructions.
- `spec/requirements/NFR-006.md`: added implementation evidence section documenting both Developer ID (macOS) and cosign keyless (all platforms) signing.

## Evidence

Keyless signing uses GitHub OIDC token; no private key stored. Signatures recorded in Sigstore Rekor transparency log. Bundle files (`.sigstore.json`) published alongside release binaries.
