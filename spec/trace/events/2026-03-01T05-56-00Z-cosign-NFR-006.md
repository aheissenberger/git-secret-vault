Added cosign keyless signing to the release workflow. A new `sign-artifacts` job runs after
build jobs, installs `sigstore/cosign-installer@v3`, and signs all `git-secret-vault` binaries
with `cosign sign-blob --yes --bundle`. Signatures uploaded as `artifacts-signatures`.
The `host` job now depends on `sign-artifacts`. Keyless signing uses GitHub OIDC token — no
private key stored. Verification instructions added to `docs/reproducible-build.md`.
