Implemented MCP (Model Context Protocol) server via `rmcp` 0.17.0 crate. The `VaultServer`
struct exposes `vault_status`, `vault_lock`, `vault_unlock`, and `vault_verify` as MCP tools
over stdio transport. Added `--mcp` flag to the root CLI that starts the JSON-RPC 2.0 server.
Compatible with Claude Desktop and other MCP clients via `VAULT_PASSWORD` env var for auth.
